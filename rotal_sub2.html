    • ROTAL은 방대한 양의 과거 및 실시간 데이터를 수집 · 가공 · 분석 · 시각화하는 고성능 산업용 빅데이터 통합 관리 솔루션입니다.
      데이터 기반 최적의 의사결정으로 업무효율을 높이고, 플랜트 운영의 혁신을 경험해보세요!

    데이터 통합

    다양한 설비 및 시스템의 데이터가 분산되어 관리되고있나요?

      1. - 여러 산업용 프로토콜 라이브러리 지원

      2. - 이기종 시스템 및 분산 데이터 통합

      3. - 초당 100만개 Time-Series Data 수집, 처리, 저장

      4. - 통신 장애 예방을 위한 게이트웨이 자체 버퍼

      5. - 서버 이중화 지원

      6. - 메모리기반의 DB로 Time-Series Data 효과적 저장 및 처리

    데이터 시각화/분석

    데이터를 충분히 이해하고 통찰력을 얻고있나요?

      1. - 여러 공정 및 데이터를 한 화면에서 한눈에 확인

      2. - 제공되는 심볼 라이브러리를 활용하여 손쉬운 편집이 가능한 공정화면 제작

      3. - 과거 특정 시간의 공정을 그대로 재연하여 비교분석 지원

      4. - 다중 타임 마커 / 상한 & 하한 제한선을 활용한 트렌드 분석

      5. - 수초안에 과거 1년 동안의 데이터 조회

      6. - 언제어디서나 웹 / 모바일 기반 데이터 차트, 공정화면 분석

    유지보수

    소중한 플랜트 설비 안전하고 효율적이게 관리하고 계신가요?

      1. - 자산 속성 / 정보 및 등록된 태그를 관리

      2. - 실시간 자산 상태를 모니터링 및 시각화

      3. - 실시간 트랜드 차트에 현재 이상 현상을 기록한 메모 등록

      4. - 특정 설비에 대한 점검 이력을 검색

      5. - 자산 경보에 따른 자산 위험순위를 조정

      6. - 실시간 자산 상태를 모니터링으로 설비 수명을 연장

    웹 모니터링

    웹으로 언제 어디서나 실시간 운영 데이터에 접근하세요

      1. - 웹 기반 24시간 프로그램 열람 및 데이터 조회가능

      2. - 토큰 기반인증으로 사용자 정보 보안 강화

      3. - SSO(Single-Sign-On)을 사용한 웹 모니터링

      4. - 모든 기기의 종류의 제한없이 조기에 경보 정보를 확인

      5. - 기존 CS버전에서 완벽하게 확인 가능한 마이그레이션

      6. - PC버전과 동등 이상의 기능 지원

      • TIC 오작동은 유틸리티의 헤더에서 수격 현상으로 이어집니다.
        현재 DCS시스템은 이 영향을 모니터링 할 수있는 기능이 없습니다.
